## Step-by-Step Setup

### Step 1: Create EmailJS Account (5 minutes)

1. Go to https://www.emailjs.com/
2. Click "Sign Up Free"
3. Create account with email/password
4. Verify email address
5. You're now logged into EmailJS dashboard

---

### Step 2: Add Gmail Service (3 minutes)

1. In EmailJS dashboard, click **"Add Service"**
2. Select **"Gmail"** from the list
3. Click **"Connect Account"**
4. Google login popup will appear
5. Enter your Gmail email (or team email)
6. Allow EmailJS to access your account
7. Back in EmailJS, you'll see:
   - **Service ID: `service_xxxxx`** (copy this)
8. Click **"Create Service"** / **"Save"**

---

### Step 3: Create Email Template (5 minutes)

1. In dashboard, click **"Email Templates"**
2. Click **"Create New Template"**
3. Set **Template Name**: `customer_query` (or any name)
4. Set **From Name**: `Lawn Mower Pro Support`
5. Set **From Email**: `{{customer_email}}`
6. Set **To Email**: `{{to_email}}`
7. Set **Reply-To Email**: `{{reply_to}}`
8. **Subject**: `New Customer Query from {{customer_name}}`
9. **Email Content** (HTML):

```html
<p>New customer query received:</p>

<table style="width: 100%; border-collapse: collapse;">
  <tr>
    <td style="padding: 10px; border: 1px solid #ddd;"><strong>Name:</strong></td>
    <td style="padding: 10px; border: 1px solid #ddd;">{{customer_name}}</td>
  </tr>
  <tr>
    <td style="padding: 10px; border: 1px solid #ddd;"><strong>Email:</strong></td>
    <td style="padding: 10px; border: 1px solid #ddd;">{{customer_email}}</td>
  </tr>
  <tr>
    <td style="padding: 10px; border: 1px solid #ddd;"><strong>Phone:</strong></td>
    <td style="padding: 10px; border: 1px solid #ddd;">{{customer_phone}}</td>
  </tr>
  <tr>
    <td style="padding: 10px; border: 1px solid #ddd;"><strong>Robot Serial:</strong></td>
    <td style="padding: 10px; border: 1px solid #ddd;">{{robot_serial}}</td>
  </tr>
  <tr>
    <td style="padding: 10px; border: 1px solid #ddd;"><strong>Message:</strong></td>
    <td style="padding: 10px; border: 1px solid #ddd;">{{message}}</td>
  </tr>
</table>

<p style="margin-top: 20px; color: #888;">
  --- This email was auto-generated by the customer support system ---
</p>
```

10. Click **"Save"**
11. You'll see: **Template ID: `template_xxxxx`** (copy this)

---

### Step 4: Get Your Public Key (2 minutes)

1. In dashboard, go to **"Account"** (top right)
2. Click **"API Keys"** or **"Public Key"**
3. Copy your **Public Key** starting with `key_` or similar

## Troubleshooting

### "Service not configured" Error

**Solution:** Check that `EMAILJS_SERVICE_ID` value is exact from EmailJS

### "Template not found" Error

**Solution:** Check that `EMAILJS_TEMPLATE_ID` matches exactly in EmailJS

### "Invalid public key" Error

**Solution:** 
- Copy entire Public Key from EmailJS
- No extra spaces
- Check it starts with `key_`

### Email not received

**Solution:**
- Check Gmail inbox AND spam folder
- Verify "From Email" in template is your Gmail
- Test in EmailJS Dashboard first ("Test It")

### Only some emails received

**Solution:**
This is normal - if one of the 4 team members' email service blocks the sender, others still receive it.
